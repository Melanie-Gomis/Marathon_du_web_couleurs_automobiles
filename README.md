# Marathon_du_web_couleurs_automobiles

Analyse des couleurs dans l'automobile

Objectif du projet :

Ce projet a pour but d’explorer et de mettre en lumière la relation entre le choix de la couleur de la voiture et différents facteurs associés.
L’objectif principal est de faire "parler les données" à travers une analyse exploratoire afin de mieux comprendre les tendances et choix de couleurs dans l’automobile.

Problématique :

Existe-t-il des tendances dans le choix des couleurs des voitures ?
Certains facteurs influencent-ils ces choix (marque, type de véhicule, période, etc.) ?

Méthodologie : 

Le projet repose sur une analyse exploratoire des données  incluant :
- Nettoyage et préparation des données
- Analyse descriptive
- Identification de tendances 

Résultats attendus :
Production de visualisations claires, lisibles et facilement interprétables.

Contraintes : 
Aucune contrainte spécifique n’a été imposée pour ce projet, laissant une grande liberté dans l’approche analytique et la visualisation.
